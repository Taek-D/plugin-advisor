       - `ALL_CATEGORIES: PluginCategory[]` — all 10 categories
       - `CONFLICT_PENALTY = 20` (per pair, within locked range -15~20)
       - `REDUNDANCY_PENALTY = 7` (per group, within locked range -5~8)
       - `UNCOVERED_PENALTY = 7` (per uncovered category — 10 x 7 = 70 max, intuitive scaling)

    3. **Helper functions (module-private):**
       - `rankForComplement(plugin: Plugin): number` — simplified trust score: verified +4, partial +1, unverified -4; beginner +3, advanced -2; stale -3, unclear -2. Used for both complement and replacement ranking.
       - `calculateScore(conflictCount, redundancyCount, uncoveredCount): number` — `Math.round(Math.max(0, Math.min(100, 100 - conflictCount * CONFLICT_PENALTY - redundancyCount * REDUNDANCY_PENALTY - uncoveredCount * UNCOVERED_PENALTY)))`
       - `buildCoverage(ids: string[]): CoverageResult` — maps ids to categories via `PLUGINS[id]?.category`, partitions ALL_CATEGORIES into covered/uncovered
       - `buildComplements(uncovered: PluginCategory[], installedIds: string[]): ComplementSuggestion[]` — for each uncovered category, find best non-installed plugin ranked by `rankForComplement`, max 1 per category. CRITICAL: filter `installedIds` to prevent recommending already-installed plugins.
       - `buildReplacements(ids: string[]): ReplacementSuggestion[]` — for each id where plugin is unverified OR partial OR stale (maintenanceStatus === "stale" maps to reason "deprecated"), find best same-category verified alternative. Return { original, reason, replacement: bestAlt?.id ?? null }.

    4. **Entry-point:**
       - `export function scorePlugins(ids: string[]): ScoringResult` — early return for empty input (empty:true, score:null, all empty arrays, all 10 categories uncovered, no complements, no replacements). Otherwise orchestrates: getConflicts -> getRedundancies -> buildCoverage -> calculateScore -> buildComplements -> buildReplacements -> return assembled ScoringResult.
