# Positioning Ideas: Plugin Advisor

*작성일: 2026-03-29*

---

## Step 1: Competitive Landscape Analysis

### Competitor 1: Claude Code Setup 플러그인 (Anthropic 공식)
- **포지셔닝**: "코드베이스를 분석해서 맞춤 자동화를 추천하는 공식 도구"
- **타겟**: Claude Code 전체 사용자
- **차별점**: Anthropic 공식, 코드베이스 자동 분석
- **빈틈**: 읽기 전용 추천만 제공. 설치 스크립트, 충돌 감지, 기존 조합 분석, 점수화 없음

### Competitor 2: awesome-claude-code (GitHub 리포)
- **포지셔닝**: "커뮤니티가 큐레이션한 Claude Code 리소스 모음"
- **타겟**: Claude Code에 관심 있는 모든 개발자
- **차별점**: 포괄적 목록, 커뮤니티 기여
- **빈틈**: 목록일 뿐 "내 상황에 뭘 써야 하는지" 판단 없음. 조합 분석/충돌 없음

### Competitor 3: MCP-tidy
- **포지셔닝**: "사용하지 않는 MCP 서버를 정리하는 도구"
- **타겟**: MCP가 축적된 중급 이상 사용자
- **차별점**: 미사용 서버 자동 탐지/정리
- **빈틈**: 정리만 가능. 추천, 분석, 충돌 감지 없음

### Competitor 4: McPick
- **포지셔닝**: "MCP 서버를 TUI로 관리하는 도구"
- **타겟**: CLI에 익숙한 개발자
- **차별점**: 터미널 기반 편리한 관리
- **빈틈**: 관리 도구일 뿐, 어떤 MCP를 선택해야 하는지 기준 미제공

### Competitor 5: 커뮤니티 블로그/YouTube
- **포지셔닝**: "내가 쓰는 플러그인 추천" (개인 경험 공유)
- **타겟**: Claude Code 초보자
- **차별점**: 실사용 경험, 스크린샷/영상
- **빈틈**: 특정 상황 기준, 최신성 보장 없음, 충돌/조합 고려 없음

## Step 2 & 3: Positioning Ideas

### Idea 1: "세팅 성공률" 포지셔닝

**Positioning Statement:**
> Plugin Advisor는 Claude Code 세팅 도구 중 유일하게 "설치 성공"에 집중하는 서비스다. 많이 추천하는 게 아니라, 첫날 바로 동작하는 조합만 추천한다.

**Strategic Rationale:**
리서치에서 가장 강한 고통은 "추천 따라했는데 안 된다"였다. 경쟁사 모두 "추천"에 집중하지만 "설치 성공 보장"을 약속하는 곳은 없다. 체크리스트, 비추천 안내, 충돌 감지는 이 약속을 뒷받침하는 구체적 기능이다.

**Supporting Messages:**
- "설치 전 체크리스트로 실패 원인을 미리 잡습니다"
- "비추천 플러그인은 이유와 함께 안내합니다"
- "충돌하는 조합은 사전에 경고합니다"

**Competitive Advantage:**
- 51개 플러그인 충돌/중복 데이터 보유 (경쟁사 없음)
- 설치 전 체크리스트 기능 (경쟁사 없음)
- 비추천 플러그인 안내 (경쟁사 없음)

---

### Idea 2: "조합 진단" 포지셔닝

**Positioning Statement:**
> Plugin Advisor는 Claude Code 플러그인의 "건강 검진 도구"다. 이미 설치된 조합을 붙여넣기만 하면 점수, 충돌, 빈틈을 진단하고 개선안을 제시한다.

**Strategic Rationale:**
경쟁사 모두 "새로 추천"에 집중하지만, "이미 깐 것 점검"을 제공하는 곳은 없다. 중급 사용자(Experimenter 페르소나)는 새 추천보다 "현재 조합이 괜찮은지" 확인이 더 급하다. 건강 검진 메타포는 친숙하고 행동을 유도한다.

**Supporting Messages:**
- "당신의 플러그인 조합 점수는 몇 점?"
- "`claude mcp list` 결과만 붙여넣으세요"
- "충돌, 중복, 빠진 카테고리를 한눈에"

**Competitive Advantage:**
- 100점 감점 모델 스코어링 (유일)
- 10개 카테고리 커버리지 분석 (유일)
- 보완/대체 플러그인 자동 추천 (유일)

---

### Idea 3: "5분 첫 세팅" 포지셔닝

**Positioning Statement:**
> Plugin Advisor는 Claude Code 첫 세팅을 5분 안에 끝내주는 도구다. 9,000+개 플러그인 중 검증된 프리셋 팩을 고르고, 설치 스크립트를 복사하면 끝.

**Strategic Rationale:**
비개발자/ChatGPT 이주자 급증 트렌드를 정면으로 겨냥. "반나절 시행착오" vs "5분 완료"의 대비가 명확하다. 시간 절약은 모든 세그먼트에 통하는 보편적 가치.

**Supporting Messages:**
- "프리셋 팩 4종: 입문자, 웹앱, 데이터, 백엔드"
- "복사 한 번이면 설치 끝"
- "체크리스트까지 확인하고 시작하세요"

**Competitive Advantage:**
- 프리셋 팩 시스템 (경쟁사 없음)
- 원클릭 설치 스크립트 생성 (경쟁사 없음)
- 난이도별 분류 (beginner/intermediate/advanced)

### Idea 4: "한국어 유일" 포지셔닝

**Positioning Statement:**
> Plugin Advisor는 Claude Code 플러그인을 한국어로 탐색하고 추천받을 수 있는 유일한 도구다. 영어 문서 해석 없이 세팅을 완성하세요.

**Strategic Rationale:**
한국 Claude Code 사용자가 빠르게 증가하지만, 한국어 전용 플러그인 추천 도구는 전무하다. IMWEB, 하이퍼리즘 등 한국 기업의 도입 사례가 나오기 시작한 시점에서, 한국어 선점은 강력한 해자가 된다.

**Supporting Messages:**
- "51개 플러그인 한국어 설명"
- "Windows MCP/npx 문제 해결 한국어 가이드"
- "한국 개발자를 위한 세팅 가이드 6종"

**Competitive Advantage:**
- 한국어 기본 UI + 영어 전환 (경쟁사 모두 영어만)
- 한국 환경(Windows) 특화 가이드
- 한국 기업 도입 사례 참조 가능

---

### Idea 5: "토큰 효율 조합" 포지셔닝

**Positioning Statement:**
> Plugin Advisor는 Claude Code 토큰 낭비를 줄이는 플러그인 조합을 추천하는 도구다. 적은 플러그인으로 더 나은 결과를 만드세요.

**Strategic Rationale:**
리서치에서 토큰 비용/낭비는 두 번째로 강한 고통이었다 ("55,000 토큰 시작 전 소모", "$500+ 월 비용"). 경쟁사 중 토큰 효율을 고려한 추천을 하는 곳은 없다. 비용 절감은 특히 유료 플랜 사용자에게 즉각적 가치.

**Supporting Messages:**
- "불필요한 MCP가 당신의 컨텍스트 윈도우를 잡아먹고 있습니다"
- "2-3개 핵심 플러그인이 10개보다 낫습니다"
- "조합 점수로 토큰 효율을 확인하세요"

**Competitive Advantage:**
- 조합 점수에 토큰 효율 반영 가능 (제품 확장 여지)
- "플러그인 수보다 세팅 성공" 철학과 일맥상통
- 경쟁사 없는 미개척 포지셔닝

## 추천 포지셔닝 전략

### Primary (핵심): Idea 1 + Idea 2 결합
**"세팅 성공을 보장하는 유일한 조합 진단 도구"**

이유: 신규 사용자(추천)와 기존 사용자(진단) 모두 포괄하며, 경쟁사가 제공하지 않는 "성공 보장 + 점수화"를 핵심으로 삼는다.

### Secondary (보조): Idea 3
**"5분 첫 세팅"**

이유: 초보자/이주자 유입 시 첫 인상으로 활용. SNS, 커뮤니티 홍보에 효과적인 구체적 숫자.

### Niche (틈새): Idea 4
**"한국어 유일"**

이유: 한국 시장 선점용. 한국 커뮤니티 타겟 마케팅에 활용.

### Future (미래): Idea 5
**"토큰 효율 조합"**

이유: 제품에 토큰 비용 표시 기능을 추가한 후 활성화. 유료 플랜 사용자 타겟.
