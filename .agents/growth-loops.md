# Growth Loops: Plugin Advisor

*작성일: 2026-03-29*

---

## 제품 핵심 가치

- **Primary action:** 프로젝트 입력 → 플러그인 추천 받기, 또는 `claude mcp list` 붙여넣기 → 조합 진단
- **Value per action:** 시간 절약 (반나절 → 5분), 안전한 세팅 확신
- **Network effects:** 약함 (개인 도구), 단 조합 점수 공유 시 바이럴 가능
- **Friction:** 낮음 (URL 접속 → 입력 → 결과, 로그인 불필요)

---

## 5가지 루프 평가

| 루프 유형 | 적합도 | 이유 |
|----------|--------|------|
| 1. Viral Loop | **4/5** | 조합 점수/결과를 SNS에 공유하면 자연 유입 발생 |
| 2. Usage Loop | **3/5** | 설치 스크립트/가이드 공유로 새 사용자 유입 가능 |
| 3. Collaboration Loop | **2/5** | 개인 도구 성격이 강해 팀 협업 루프는 약함 |
| 4. User-Generated Loop | **3/5** | 플러그인 제안 기능으로 커뮤니티 기여 가능 |
| 5. Referral Loop | **2/5** | 무료 도구라 보상 설계가 어려움 |

## Primary Loop: Viral Loop — "조합 점수 공유"

```
사용자가 조합 분석 실행
    ↓
점수 + 결과 화면 스크린샷/공유
    ↓
X/커뮤니티에서 "내 점수는 XX점" 포스팅
    ↓
다른 사용자가 "나도 해볼까?" → 사이트 방문
    ↓
새 사용자가 조합 분석 실행
    ↓ (반복)
```

### 구현 방안

**Step 1: 공유 유도 (마찰 최소화)**
- 조합 분석 결과 화면에 "공유하기" 버튼 추가
- 클릭 시 미리 작성된 텍스트 + 결과 요약 복사
- 예: "내 Claude Code 플러그인 조합 점수: 85/100 🎯 충돌 0개, 빠진 카테고리 2개 → plugin-advisor.vercel.app"

**Step 2: 공유 콘텐츠의 매력도**
- 점수를 시각적으로 표현 (게이지, 등급 배지)
- "A+ / A / B / C / D" 등급 시스템으로 자랑/개선 동기 부여
- 높은 점수 → "자랑하고 싶음", 낮은 점수 → "개선하고 다시 해보고 싶음"

**Step 3: 랜딩 최적화**
- 공유 링크로 들어온 사용자에게 "당신의 점수도 확인해보세요" CTA
- 로그인 없이 바로 분석 가능 (마찰 제로)

### Loop Coefficient 추정
- 사용자당 공유: 10%가 공유한다고 가정
- 공유당 노출: 평균 50명 (X 기준)
- 노출→방문 전환: 5%
- 방문→사용 전환: 30%
- **1,000명 사용 → 100명 공유 → 5,000 노출 → 250 방문 → 75 신규 사용**
- **Loop coefficient: 0.075** (자체 성장은 느리지만, 콘텐츠 마케팅과 결합 시 증폭)

## Secondary Loop: Usage Loop — "가이드/스크립트 공유"

```
사용자가 추천 결과 + 설치 스크립트 받음
    ↓
블로그/Slack/팀 채널에 "이 스크립트로 세팅했다" 공유
    ↓
동료/팔로워가 링크 따라 방문
    ↓
새 사용자가 같은 프리셋 팩 또는 맞춤 추천 사용
    ↓ (반복)
```

### 구현 방안
- 설치 스크립트에 "# Generated by Plugin Advisor" 주석 포함
- 추천 결과를 마크다운으로 내보내기 기능 (팀 공유용)
- 가이드 페이지 하단에 "이 가이드가 도움이 됐다면 공유해주세요" CTA

## Tertiary Loop: User-Generated Loop — "플러그인 제안 → DB 확장"

```
사용자가 새 플러그인을 제안
    ↓
관리자 검토 → DB에 추가
    ↓
DB가 풍부해짐 → 추천 품질 향상
    ↓
더 많은 사용자 유입 + 만족도 향상
    ↓
더 많은 사용자가 플러그인 제안
    ↓ (반복)
```

### 구현 방안
- 이미 존재: 플러그인 제안 기능 (/plugins 하단)
- 강화: 제안이 반영되면 기여자 표시 (인정 인센티브)
- "이 플러그인이 없네요? 제안해주세요" CTA 강화
