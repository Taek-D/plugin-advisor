# Wireframe — Claude Code Plugin Advisor

## 화면 구조

---

## Screen 1: 입력 화면 (/)

```
┌─────────────────────────────────────────────────────┐
│  ● PLUGIN ADVISOR          [Claude Code · 10 plugins]│
├─────────────────────────────────────────────────────┤
│                                                     │
│  어떤 걸 만들고 있나요?                               │
│  PRD, README, 프로젝트 설명을 넣으면...               │
│                                                     │
│  [텍스트 입력] [파일 업로드] [GitHub URL]             │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │ 프로젝트를 설명해줘요.                        │   │
│  │ 예) React 기반 SaaS 대시보드...              │   │
│  │                                             │   │
│  │                                             │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  [    ANALYZE → RECOMMEND    ]                      │
│                                                     │
│  [OMC] [BKIT] [SP] [UI] [C7] [RALPH] ...           │
│  ↑ 태그 클릭하면 플러그인 상세 볼 수 있어요           │
└─────────────────────────────────────────────────────┘
```

**입력 방식 탭 전환:**
- 텍스트: textarea 직접 입력
- 파일: drag & drop 또는 클릭 업로드 (.md, .txt)
- GitHub: URL input → 서버에서 README fetch
