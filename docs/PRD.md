# PRD — Claude Code Plugin Advisor

**버전**: v1.0  
**작성일**: 2026.03.05  
**상태**: 개발 진행 중

---

## 1. 개요

### 서비스 한 줄 정의
PRD / README / GitHub URL을 입력하면 최적의 Claude Code 플러그인 조합을 추천하고, 설치 스크립트를 생성해주는 웹 서비스

### 배경 & 문제
- Claude Code 플러그인 생태계가 빠르게 성장 중 (9,000개 이상)
- 어떤 플러그인을 쓸지, 뭐랑 같이 쓰면 되는지 정리된 곳이 없음
- 유저들이 이것저것 설치했다가 충돌 → 다 지우고 포기하는 패턴 반복
- 직접 겪은 페인포인트 기반

### 타겟 유저
- Claude Code를 처음 시작하는 개발자
- 바이브코딩으로 새 서비스/게임 개발을 시작하려는 인디 개발자
- 플러그인 조합을 최적화하고 싶은 기존 Claude Code 유저

## 2. 핵심 기능

### P0 (MVP 필수)
- [ ] 텍스트 직접 입력으로 프로젝트 설명
- [ ] 파일 업로드 (.md, .txt)
- [ ] GitHub URL로 README 자동 불러오기
- [ ] 키워드 기반 플러그인 추천 (10개 플러그인 DB)
- [ ] 추천 이유 + 매칭 키워드 하이라이트 표시
- [ ] 충돌 경고 (OMC ↔ Superpowers 등)
- [ ] 플러그인 상세 모달 (설명, 기능, GitHub 링크)
- [ ] 설치 스크립트 생성 & 클립보드 복사
- [ ] 체크박스로 원하는 것만 선택

### P1 (배포 후 추가)
- [ ] 플러그인 DB 확장 (현재 10개 → 30개+)
- [ ] 분석 히스토리 저장 (localStorage)
- [ ] 플러그인 조합 즐겨찾기
- [ ] 커뮤니티 추천 조합 공유
- [ ] 플러그인 최신 버전 알림

### P2 (장기)
- [ ] AI 기반 분석 (Claude API 옵션)
- [ ] 플러그인 리뷰 & 평점
- [ ] 다국어 지원 (EN/KO)

## 3. 기술 스택

### 프론트엔드
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + shadcn/ui
- **Language**: TypeScript
- **State**: React useState (클라이언트 상태)

### 배포
- **Hosting**: Vercel
- **Domain**: 미정

### 데이터
- 플러그인 DB: 정적 JSON/TS 파일 (현재)
- 추후 Supabase 마이그레이션 고려

---

## 4. 비기능 요구사항

- API 호출 없이 즉시 분석 (키워드 기반)
- GitHub URL fetch는 서버사이드에서 처리 (CORS 우회)
- 모바일 대응 (반응형)
- 초기 로딩 1초 이하

---

## 5. 페이지 구조

```
/                   → 메인 (입력 → 결과 SPA)
/plugins            → 전체 플러그인 목록 (P1)
/plugins/[id]       → 플러그인 상세 (P1)
```
